## conda命令
```
# 创建环境
conda create -n ml python=3.9
# 激活环境
conda activate ml
# 查看环境列表
conda env list
# 查看当前环境下用conda安装的库
conda list
```
环境搭建 参考 https://www.cnblogs.com/yangshifu/p/17071889.html
  
## 数据处理流程  
1. 导入数据
2. 选取需要的列和行
3. 数据归一化，减去平均值、除以标准差
4. 拆分数据 训练集、验证集、测试集
5. 配置模型 进行训练
